# Eggsploit
A YSWS program associated with PullQuests (for Clubs). Participants find bugs and easter eggs hidden by their club leader, send a PR to fix the bugs and include the easter eggs in the PR description.

##
This is currently running in vite:
### Local Development
```bash
npm install
npm run dev
```
(OR)
```bash
npm install
npm run devhost
```
to run it locally and access via other devices
### Build For Vercel

```bash
npm run build
```

Note - This website has audio files (which might be loud) so please reduce your volume :)
