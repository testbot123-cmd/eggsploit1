# ByteBrew - Eggsploit Demo 

A whole demo kinda site (coffee shop website for developers). Every drink is named after a programming concept. Built for Eggsploit :))

There are two versions obviously:
- `broken/` has intentional bugs you need to find and fix -> given to clubs
- `fixed/` is the clean version with all bugs fixed -> what is expected to be done by clubs

Easter eggs are in both versions.
